# Blush & Buff Nail Studio Sample Assets

This package contains simplified sample assets for a fictitious nail salon website build.

The goal is to test a low-cost website workflow using a realistic small-business asset set. This package does not include generated logo files or generated photos. Those should be created separately using the Nano Banana prompts and then added to the `logo/` and `images/` folders.

## Folder Structure

- `ai-tester-builder-brief.md` - prompt for the AI website builder
- `copy/website-copy.md` - primary website copy
- `copy/about-sanitation-gifts-parking-loyalty.md` - supplemental website copy
- `data/business-info.json` - business name, contact details, hours, and walk-in policy
- `data/booking-online-presence.json` - included and excluded booking / online presence links
- `data/services-menu.json` - structured services and pricing data
- `data/team.json` - sample technician information
- `menu/blush-and-buff-services-menu.pdf` - sample services and pricing PDF
- `prompts/nano-banana-logo-prompt.md` - prompt for creating the logo
- `prompts/nano-banana-photo-prompts.md` - prompts for creating 14 photo assets
- `images/README.md` - instructions for adding generated photos
- `logo/README.md` - instructions for adding the generated logo

## Included Online Presence Links

- Google Business Profile placeholder URL
- Instagram handle and placeholder URL

## Excluded Links

- Booking system link
- Facebook page URL
- Yelp profile URL

## Notes

All URLs are sample placeholders. They should be wired into the test website exactly as provided, but they should not be visited or verified.

The services menu is provided as a PDF to match the real-world expectation that a small nail salon may already have a price list or menu file.
